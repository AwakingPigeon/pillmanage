# 抑郁服药助手 - MVP文档

## 项目定位
**专为抑郁症患者设计的极简服药提醒应用**

## MVP核心功能（仅3个）

### 1. 服药提醒 ⏰
- **功能**：每日定时提醒服药
- **交互**：简单设置时间，一键确认服药
- **技术**：本地通知，无需网络

### 2. 服药记录 ✅
- **功能**：记录每日是否服药
- **交互**：服药后点击确认，自动记录
- **数据**：仅保存"是/否"状态

### 3. 药物管理 💊
- **功能**：添加/删除药物
- **交互**：输入药名即可，无需复杂信息
- **限制**：最多管理3种药物

## 明确砍掉的功能 ❌
- ❌ AI聊天助手
- ❌ 复杂剂量计算
- ❌ 药物相互作用检查
- ❌ 云端同步
- ❌ 多设备支持
- ❌ 社交分享
- ❌ 医生沟通
- ❌ 情绪追踪

## 用户流程（极简）
1. **首次打开** → 设置一种药物 → 设置提醒时间
2. **每日使用** → 收到通知 → 点击"已服药" → 完成
3. **查看记录** → 查看本周服药情况

## 技术实现
- **框架**：React Native（跨平台）
- **存储**：AsyncStorage（本地存储）
- **通知**：expo-notifications
- **页面**：仅4个页面
  - 首页（今日提醒）
  - 记录页面
  - 药物列表
  - 设置页面

## 数据模型（极简）
```javascript
// 药物
{
  id: string,
  name: string,
  reminderTime: string // "09:00"
}

// 服药记录
{
  date: string, // "2025-11-12"
  taken: boolean // true/false
}
```

## UI设计原则
- **大按钮**：抑郁状态下易于点击
- **高对比度**：视觉清晰
- **无干扰**：纯色背景，无动画
- **正面反馈**：服药后显示鼓励文字

## 开发时间估算
- **总时间**：3-5天
- **第1天**：项目搭建 + 药物管理
- **第2天**：提醒功能
- **第3天**：记录功能
- **第4-5天**：测试优化

## 成功指标
- **功能完成度**：3个核心功能100%可用
- **用户体验**：3步内完成日常操作
- **稳定性**：7天无崩溃
- **存储安全**：数据100%本地保存
