# 抑郁服药助手 - MVP文档

## 项目定位
**专为抑郁症患者设计的极简服药提醒应用**

## MVP核心功能（仅3个）

### 1. 服药提醒 ⏰
- **功能**：根据预设服药计划，固定时间提醒服药，支持隔日提醒
- **交互**：设置一次时间，每日自动提醒
- **技术**：本地通知，无需网络

### 2. 服药记录 ✅
- **功能**：简单记录是否服药
- **交互**：提醒后，用户服药，滑动"已服用"，记录当天状态
- **数据**：记录具体时间

### 3. 药物管理 💊
- **功能**：提醒用户及时补药，避免忘记买药造成的戒断反应。
- **交互**：在设置服药计划「药物名；数量单位片 如 几片，一片，半片，四分之一片；提醒时间」之外记录现有药物库存数量，多少片。在药物用完前三天提醒，这个三天支持自定义设置
- **限制**：仅支持一种药物（MVP阶段）

## 用户流程（极简）
1. **首次打开** → 选择预设药物或者输入其他药物名称 → 选择服药剂量（小数） → 设置服药时间（提醒时间）→ 设置药品库存数量
2. **每日使用** → 收到通知 → 滑动"已服用"（2步完成）
3. **查看记录** → 药物用完前三天提醒补药（支持自定义提前时间）

## 技术实现
- **框架**：React  Native（跨平台）
- **存储**：AsyncStorage（本地存储）
- **通知**：expo-notifications
- **页面**：仅4个页面
  - 首页（今日提醒）
  - 记录页面
  - 药物列表
  - 设置页面

## 数据模型（极简）
```javascript
// 仅一个药物配置
{
  medicationName: string,    // "抗抑郁药"
  reminderTime: string,      // "09:00"
  isActive: boolean          // true
}

// 服药记录（简化版）
{
  "2025-11-12": true,        // 已服药
  "2025-11-13": false        // 未服药
}
```

## UI设计原则
- **大按钮**：便于躯体化特别是手抖点击
- **高对比度**：视觉清晰
- **无干扰**：纯色背景，无动画
- **正面反馈**：服药后显示鼓励文字
