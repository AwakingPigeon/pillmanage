# 抑郁服药助手 - MVP分支

这是一个**真正的MVP分支**，仅包含MVP开发所需的最小文件集。

## 分支内容
- `MVP.md` - 完整的MVP文档和功能规格
- `package.json` - 最小化的依赖配置
- `README.md` - 本说明文档

## 使用方法
1. 基于此分支创建新的开发分支
2. 逐步添加MVP所需的代码文件
3. 保持分支干净，仅包含MVP功能

## 不包含的内容
- 任何源代码文件
- 复杂的配置文件
- 非必要的依赖
- 测试文件
- 文档（除MVP文档外）

## 开发建议
- 从空白开始，逐步构建MVP功能
- 每个功能完成后及时提交
- 保持代码简洁，专注于核心需求

## 目标用户
专为抑郁症患者设计的极简服药提醒应用。
