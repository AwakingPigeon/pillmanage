# 抑郁服药助手 - MVP文档

## 🎯 项目定位
**一句话描述**：专为抑郁症患者设计的极简服药提醒应用

## 🚀 MVP核心功能（80/20法则）

### 1. 必须有的功能（20%功能解决80%问题）
- **服药提醒** - 每天固定时间推送
- **服药记录** - 一键确认已服药
- **药物管理** - 添加/修改药物信息

### 2. 可以砍的功能（后续版本再加）
- ❌ 复杂的心情记录
- ❌ AI聊天功能  
- ❌ 多设备同步
- ❌ 药品图片识别
- ❌ 社交分享功能

## 📱 用户流程（极简版）

### 新用户首次使用
1. 打开App → 添加药物 → 设置服药时间 → 完成
2. 每天收到提醒 → 点击"已服药" → 记录完成

### 日常用户
1. 查看今日是否已服药
2. 如需修改，进入药物管理

## 🏗️ 技术实现（最小可行）

### 技术栈
- **React Native** (跨平台，一套代码)
- **本地存储** (AsyncStorage，无需后端)
- **本地通知** (Expo Notifications)

### 页面结构（4个页面）
```
App/
├── 首页/今日状态      # 显示今日服药状态
├── 药物管理          # 添加/编辑药物
├── 设置提醒时间      # 设置每天提醒时间
└── 历史记录          # 查看服药历史
```

### 数据模型（极简）
```javascript
// 药物信息
{
  id: 1,
  name: "舍曲林",
  dosage: "50mg",
  time: "09:00",  // 每天固定时间
  isActive: true
}

// 服药记录
{
  date: "2024-01-15",
  taken: true     // 是否已服药
}
```

## 🎨 UI设计原则

### 设计要点
- **大按钮** - 抑郁症患者可能手抖
- **高对比度** - 视觉清晰
- **简单文案** - 减少认知负担
- **无干扰** - 去掉一切不必要元素

### 颜色方案
- **主色**: 深蓝色 (#2563EB) - 稳定、信任
- **成功**: 绿色 (#10B981) - 已服药
- **警告**: 橙色 (#F59E0B) - 未服药提醒

## 📊 开发时间估算

### 开发周期：2-3周
- **第1周**: 基础框架 + 药物管理
- **第2周**: 提醒功能 + 服药记录
- **第3周**: 测试优化 + 发布准备

### 人力需求
- **1名React Native开发者** (全职)
